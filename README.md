# Watchdog

## Roadmap

- [ ] Notifications about detected motion
- [ ] Push notifications
- [ ] Web rtc video transfer
- [ ] Sound detection
- [ ] Remote configuration
- [ ] Really good detection algos
- [ ] Easy way to host
- [ ] Screenshots saving
- [ ] Video saving
- [ ] WebRTC through server (TURN)

## Usage

Install (OSX)

```
bash install.sh
```

Run

```
npm start
```
